pass_arena2

The map pass_arena2 created by Jaguar for the Team Fortress 2 4v4 PASS Time community. This was the first map ever made for the gamemode, both of which Jaguar created. Files that start with goalfix_ are likely created by EaasyE, the rest created by Jaguar. Exer took over development for version b8 onwards, since Jaguar did not develop the map anymore. Danmar also has contributed via mainly decoration passes. The listings for maps below are arranged in their most likely order chronologically, but their true order is unknown at this time.
