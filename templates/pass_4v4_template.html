pass_4v4_template

The uncompiled map made by Exer for the Team Fortress 2 4v4 PASS Time community. It is intended as template for mappers to use to more easily create 4v4 PASS Time maps, since the logic can be overwhelming to those unfamiliar with PASS Time's map logic.
